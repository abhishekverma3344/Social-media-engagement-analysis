# Social Media Engagement Analysis

This project analyzes user behavior on an Instagram-like social media platform using **MySQL** and **Excel visualizations**.  
The goal is to understand **which users create the most engagement**, which users are inactive, which **hashtags perform best**, and what **posting times drive the highest interactions**.
